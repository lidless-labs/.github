# Lidless Labs

Constant Vigilance.

Lidless Labs builds open-source SOC, network, and homelab tooling for MCP clients and AI-assisted operations. The work is local-first, MIT licensed where possible, and shaped around real systems that have to be queried under pressure.

## Security / SOC

- [wazuh-mcp](https://github.com/lidless-labs/wazuh-mcp) - Wazuh SIEM/XDR integration for MCP clients.
- [misp-mcp](https://github.com/lidless-labs/misp-mcp) - MISP threat intelligence lookup and enrichment.
- [suricata-mcp](https://github.com/lidless-labs/suricata-mcp) - Suricata EVE JSON alert analysis and rule workflows.
- [thehive-mcp](https://github.com/lidless-labs/thehive-mcp) - TheHive case, alert, task, and observable operations.
- [cortex-mcp](https://github.com/lidless-labs/cortex-mcp) - Cortex observable analysis and response workflows.
- [mitre-mcp](https://github.com/lidless-labs/mitre-mcp) - MITRE ATT&CK mapping, enrichment, and detection-gap support.
- [zeek-mcp](https://github.com/lidless-labs/zeek-mcp) - Zeek log parsing, querying, and network security analysis.

## Network

- [librenms-mcp](https://github.com/lidless-labs/librenms-mcp) - LibreNMS devices, ports, alerts, acknowledgements, and maintenance windows.
- [n8n-ops-mcp](https://github.com/lidless-labs/n8n-ops-mcp) - n8n workflow inspection, validation, execution, and ops automation.

## Homelab

- [proxmox-mcp](https://github.com/lidless-labs/proxmox-mcp) - Proxmox VE read and safe-write tools for MCP clients.
- [adguard-mcp](https://github.com/lidless-labs/adguard-mcp) - AdGuard Home DNS, filtering, rewrite, and client management.
- [immich-mcp](https://github.com/lidless-labs/immich-mcp) - Immich photo library search, albums, memories, people, and duplicate workflows.
